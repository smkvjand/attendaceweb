Firebase Setup

Same database as your Android app — no migration needed

This website uses the exact same Firebase project as your Android app — same Firestore, same Auth, same data. Just paste the config and it works.
Connect Firebase
1

Open Firebase Console

Go to console.firebase.google.com → open your project.

2

Register Web App

Project Settings → Your apps → click </> → nickname → Register → copy the firebaseConfig.

3

Paste into this file

Find PASTE YOUR FIREBASE CONFIG HERE in index.html and replace:

const firebaseConfig = { apiKey: "AIza...", projectId: "your-app", ... };
4

Update Firestore Rules

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }
5

Add authorized domain

Authentication → Settings → Authorized domains → Add your GitHub Pages domain: your-username.github.io

Deploy to GitHub Pages
1

Create repo

github.com → New → name: attendease → Public → Create.

2

Upload file

Add file → Upload → drag index.html → Commit.

3

Enable Pages

Settings → Pages → Source: main → / (root) → Save.

4
