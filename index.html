Firebase Setup Guide
Connect this website to your existing Firebase project — same database as your Android app
This website uses the exact same Firebase project as your Android app — same Firestore database, same Auth users, same data. No migration needed.
Step-by-step connection
1

Open your Firebase project

Go to console.firebase.google.com and click on the same project you used for the Android app.

2

Add a Web App (if not already done)

Project Settings → Your apps → click the Web icon </> → give it a nickname like "AttendEase Web" → Register app → copy the firebaseConfig object.

3

Paste config into js/firebase-config.js

Open the js/firebase-config.js file and replace the placeholder values:

const firebaseConfig = { apiKey: "AIza...", authDomain: "your-app.firebaseapp.com", projectId: "your-app-id", storageBucket: "your-app.appspot.com", messagingSenderId: "123456789", appId: "1:123...:web:abc" };
4

Enable Email/Password Auth (already done for app)

Firebase Console → Build → Authentication → Sign-in method → Email/Password should already be enabled. If not, enable it and save.

5

Update Firestore Rules to allow web access

Build → Firestore → Rules tab → make sure the rules allow authenticated users:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }

Click Publish.

6

Add your GitHub Pages domain to Auth

Firebase Console → Authentication → Settings → Authorized domains → Add domain → paste your GitHub Pages URL:

your-username.github.io

This allows sign-in from your hosted site.

Deploy to GitHub Pages
1

Create a GitHub repo

Go to github.com → New repository → name it attendease → Public → Create.

2

Upload all files

Drag and drop all the project files (index.html, css/, js/) into the repo. Commit directly to main.

3

Enable GitHub Pages

Repo → Settings → Pages → Source: Deploy from a branch → Branch: main → folder: / (root) → Save.

4

Your site is live!

In 1–2 minutes your site is at:
https://your-username.github.io/attendease
